# Documentation

This directory turns the hook pack into task-oriented documentation. Start
with the page that matches what you are trying to do.

- [First install tutorial](tutorial-first-install.md) walks through a dry run,
  hook copy, settings merge behavior, and rollback.

## Solve a task

- [How-to guide](how-to.md) covers common maintenance work: refresh a source

  uninstall cleanly.

## Look something up

- [Reference](reference.md) documents repository layout, scripts, hook lifecycle
  wiring, installed files, and verification commands.

## Understand the design

  backed, why the settings fragment can be empty, and why hooks default to
